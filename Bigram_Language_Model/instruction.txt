1) Для Прочитать данных с файла в структуры данных использован файл "bigram_table.py".Задача была выполнена с использованием файлов JSON.Результат вы можете увидеть в файле "bigram.xlsx".

2)"calculating_probabilities.py" файл использовался для измерения вероятностей.Результат вы можете увидеть в файле "probabilities.xlsx".

3)Файл "gererate_name.py" использовался для создания новых имен.Данный код использует библиотеку PyTorch и модуль json для генерации случайного имени длиной 6 символов на основе списка всех возможных биграмм, хранящихся в файле 'all_bigrams.txt'.

4)Визуализация таблицы в картинку creat_main_graph.py". Вы можете увидеть изображение "graph_sample.png" в качестве примера или запустить код.

5)Нейронная сеть в файле "neuroset.py".
Данный код создает и обучает модель глубокого обучения (нейронную сеть) для предсказания вероятности следующей биграммы на основе частотности биграмм в тексте, который находится в файле 'prob_freq.txt'.
Конкретно, код загружает данные о частотности биграмм из файла 'prob_freq.txt' и создает два списка: список биграмм и список соответствующих им вероятностей. Затем код создает входные и выходные данные для обучения модели: входные данные - это коды символов, соответствующие каждой биграмме, а выходные данные - это вероятности соответствующих биграмм.
